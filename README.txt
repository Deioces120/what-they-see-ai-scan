# 🔍 what-they-see-ai-scan (v1.0)

> A Chrome extension powered by AI to scan and visualize the security status of any website — instantly.
---------------------------------------------------------------------------------------------------------------------------------------

## 🚀 Features

- ✅ AI-based security scan of the current website
- 🎨 Beautiful and user-friendly UI
- 🔐 Highlights both security issues and trusted points
- 💡 Works instantly — no setup or login required

## 🧩 How to Install

1. Download or clone this repository
2. Go to chrome://extensions/ in your Chrome browser
3. Enable Developer mode (top right)
4. Click on "Load unpacked"
5. Select the project folder

That's it! You’ll see the extension icon appear in your toolbar.

---------------------------------------------------------------------------------------------------------------------------------------


## 💬 Why This Project?

Most users visit websites without knowing what's really going on behind the scenes.  
This extension helps anyone — even non-tech people — to get a clear view of how safe a page is, using the power of AI.
